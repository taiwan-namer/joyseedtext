# 總站列表課配對碼（`listing_bind_token`）

- **新增**：在總站後台新增課程時，若所屬商家為本站的 `NEXT_PUBLIC_CLIENT_ID`，可勾選「代銷列表課：建立時自動產生配對碼」（未填庫存綁定、未手填老師配對碼時才會寫入欄位）。若 DB 尚未有該欄位，會自動略過寫入並重試，不會回傳假陽性。
- **編輯**：編輯總站列表課時可使用「一鍵綁定老師課程」（選老師店家與課程後執行），等同寫入列表課 `inventory_*` 與老師課 `hq_listing_*`。

詳細手動／老師填碼流程見 `HQ_MARKETPLACE_TEACHER_BINDING.md`。
